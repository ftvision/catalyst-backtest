# Schemas

Language-neutral contracts for the system.

Start with JSON Schema files here, then keep Python Pydantic models and Rust
Serde structs aligned with these contracts.

Planned schemas:

- `graph.schema.json`
- `backtest-request.schema.json`
- `backtest-result.schema.json`
- `market-data-bundle.schema.json`
- `simulation-trace.schema.json`
- `simulation-policy.schema.json`
